How the work process looks

1. Documentation

You gather the company's invoices, receipts, and other documents every month.

2. Send

You send the documents via an app that we work with. On the app, you can directly upload files or take pictures.

3. We bookkeep

Once the documents are uploaded, we bookkeep all business events and ensure that declarations are submitted on time.

4. Communication

If something is missing, we contact you via phone or email. If everything is in order, we complete the work and update you with receipts and reports.

5. Save documentation

By law, documentation must be saved for 7 years, this is done in our accounting program.
